# ollie-webbluetooth
Simple application to control the Sphero Ollie with web bluetooth

# Demo

You can test the application here : https://jef.binomed.fr/sphero_ollie-web-bluetooth/

# What you can do ?

You can : 
* Move the Ollie
* Change the light
* Do some tricks

What's comming next : 
* Control with a "circle interface" instead of buttons
